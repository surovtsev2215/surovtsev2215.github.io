Хаб ПТО (Мои_утилиты)

1. Установите Python 3.10+ с tkinter (на Windows обычно входит в установку python.org).

2. Зависимости ядра (обязательно, если не ставили раньше):

   pip install -r requirements.txt

   Запускайте команду из папки «Мои_утилиты» (рядом с этим файлом).

3. Дополнительные пакеты отдельных утилит — см. requirements.txt в папке каждой утилиты и её УСТАНОВКА.txt.

4. Отладочные логи хаба (по желанию): задайте переменную окружения PTO_DEBUG=1 и перезапустите хаб.
   В корне репозитория появится файл debug-pto-hub.jsonl (он в .gitignore).
