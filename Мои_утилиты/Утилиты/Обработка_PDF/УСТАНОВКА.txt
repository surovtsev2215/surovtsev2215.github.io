Утилита «Обработка PDF»
========================

Python-пакеты (один раз), из папки этой утилиты:

  pip install -r requirements.txt

Или из корня проекта:

  pip install -r "Мои_утилиты/Утилиты/Обработка_PDF/requirements.txt"


Tesseract OCR (для распознавания сканов)
-----------------------------------------

1. Установите Tesseract для Windows (официальный инсталлятор или пакет через winget/chocolatey).
2. При установке отметьте языковые данные: Russian, English.
3. Убедитесь, что в PATH доступна команда `tesseract`, либо задайте путь через переменную
   окружения (см. документацию pytesseract), если утилита не находит движок.

Без Tesseract работает извлечение обычного текста из PDF; пустые отсканированные страницы
останутся без текста (с предупреждением).


Таблицы
-------

Опционально используется pdfplumber (уже в requirements.txt). Если пакет не установлен,
флажок «Таблицы» не даст результата до установки зависимостей.


Перезапуск хаба после установки пакетов.
