Как загружать КС-2 через чат без повторной загрузки Excel

1) Отправьте файл КС-2 в чат.
2) Попросите: «прочитай КС-2 и подготовь JSON для импорта в программу».
3) Сохраните полученный JSON на диск.
4) В утилите «Выполнение» нажмите кнопку «Импорт результата JSON…».
5) Выберите JSON-файл — данные попадут в локальное хранилище.

Важно:
- После импорта данные хранятся в `Данные/ks2_results_store.pkl`.
- При следующем запуске программы Excel повторно загружать не нужно.
- Повторный импорт того же документа не создаёт дубль (используется content hash).

Резервный вариант (без чата):
- Запустите `конвертер_кс2_json.py`, чтобы получить JSON из Excel.
- Затем импортируйте JSON той же кнопкой.
