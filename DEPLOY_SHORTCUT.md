# Ярлык автодеплоя сайта

## Что делает `deploy-site.bat`

Скрипт в корне проекта выполняет:

1. Проверку, что запуск из git-репозитория.
2. Проверку изменений (`git status --porcelain`).
3. `git add .` и авто-коммит, если есть изменения.
4. Сборку фронтенда (`web/frontend`, `npm run build`).
5. `git push` (кроме dry-run режима).
6. Финальный `git status --short`.

Все шаги пишутся в лог: `deploy-last.log` в корне проекта.

## Как создать ярлык на рабочем столе

1. Откройте папку `C:\Users\Surovcev\PTO_Project`.
2. Найдите файл `deploy-site.bat`.
3. Нажмите правой кнопкой мыши -> `Отправить` -> `Рабочий стол (создать ярлык)`.
4. Переименуйте ярлык, например в `Обновить сайт`.

## Запуск

- Обычный режим: двойной клик по ярлыку.
- Тест без `git push`:
  - Откройте терминал в корне проекта.
  - Запустите: `deploy-site.bat --dry-run`

## Если что-то пошло не так

1. Откройте файл `deploy-last.log`.
2. Найдите последнюю строку `ERROR: ...`.
3. Исправьте причину и запустите ярлык снова.
